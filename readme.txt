Easily record Looms within your Wordpress editor!

== Description ==

Use the Loom Record Wordpress block to record and embed a Loom directly into your Wordpress site! 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/record-loom` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Why does it say `unable to record`? =

The Loom Record block depends on the MediaRecorder API and enablement of third-party cookies so it will not work in all browsers.
